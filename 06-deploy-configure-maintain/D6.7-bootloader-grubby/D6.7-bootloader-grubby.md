# KP D6.7 — Modify the System Bootloader (grubby / grub2-mkconfig)
**6 tasks · RHEL 10 · server1 as root**

**1.** Edit `/etc/default/grub` so that: `GRUB_TIMEOUT=10`, add the line `GRUB_TIMEOUT_STYLE=hidden`, and add `quiet` to the **end** of the `GRUB_CMDLINE_LINUX` line. Then regenerate the GRUB configuration.

**2.** Add a kernel argument (e.g. `audit=1`) to **all** installed kernels using `grubby`.

**3.** **Remove** a kernel argument from all kernels using `grubby`.

**4.** Show the default kernel's current arguments (`grubby --info=DEFAULT`) and save them to `/root/kargs.txt`.

**5.** After rebooting, verify a kernel argument actually took effect by checking `/proc/cmdline`.

**6.** State the correct path to regenerate the GRUB config on RHEL 10 (for both BIOS and UEFI) and run the command.
