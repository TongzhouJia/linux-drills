# KP D6.3 — Services: enable / disable / status at boot
**5 tasks · RHEL 10 · server1 as root**

**1.** Ensure `httpd` is **enabled at boot and started now** using a single command.

**2.** **Stop and disable** a given service, then confirm it is neither running nor set to start at boot.

**3.** **Mask** a service so it cannot be started at all, then **unmask** it.

**4.** List all **failed** services and save them to `/root/failed.txt`.

**5.** Check whether `firewalld` is **enabled at boot** (`is-enabled`) and save the answer to `/root/enabled.txt`.
