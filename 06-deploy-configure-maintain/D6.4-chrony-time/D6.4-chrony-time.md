# KP D6.4 — Time Service Client (chrony)
**6 tasks · RHEL 10 · server1 as root**

**1.** Confirm the `chrony` package is installed and the `chronyd` service is running.

**2.** Configure `chronyd` to synchronize with **`us.pool.ntp.org`**, and apply it.

**3.** Show the current time sources with `chronyc sources` and save the output to `/root/sources.txt`.

**4.** Set the system timezone to **`Asia/Shanghai`**.

**5.** Ensure NTP synchronization is enabled via `timedatectl` and verify it is active.

**6.** Show the synchronization status (`chronyc tracking`) and save it to `/root/tracking.txt`.
