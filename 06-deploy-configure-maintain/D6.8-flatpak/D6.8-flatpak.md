# KP D6.8 — Flatpak (RHEL 10 new objective)
**6 tasks · RHEL 10 · server1 as root**

**1.** Add the **Flathub** remote **system-wide**.

**2.** List the configured flatpak remotes and save them to `/root/remotes.txt`.

**3.** Search for an application (e.g. `gimp`) in the remotes.

**4.** Install an application **system-wide** (using `sudo`).

**5.** List installed flatpak apps and save to `/root/flatpaks.txt`, noting the **system vs user** installation column.

**6.** Uninstall the flatpak application you installed.
