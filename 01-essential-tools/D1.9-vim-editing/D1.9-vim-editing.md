# KP D1.9 — Text Editing with vim
**5 tasks · RHEL 10 · server1 as root · use vim for each**

**1.** Using vim, create `/root/vimtest.txt` containing exactly three lines — `one`, `two`, `three` — then save and quit.

**2.** Open `/root/vimtest.txt` in vim and replace **every** occurrence of the letter `o` with `0` across the whole file, then save.

**3.** Open `/etc/hosts` in vim in **read-only** mode (so you cannot accidentally change it) and quit.

**4.** Without leaving vim, append the current date to the bottom of `/root/vimtest.txt` by running the `date` command from inside vim, then save.

**5.** Create `/root/lines.txt` with 6 lines (`line1`…`line6`), then in vim delete lines **2 through 4** and save.
