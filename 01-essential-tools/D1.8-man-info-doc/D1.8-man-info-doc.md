# KP D1.8 — System Documentation (man / info / /usr/share/doc)
**5 tasks · RHEL 10 · server1 as root**

**1.** The word `passwd` has both a **command** man page and a **file-format** man page. Write into `/root/mananswer.txt` the exact `man` command you would run to read the **file-format** (the `/etc/passwd` layout) page.

**2.** Use `man -k` / `apropos` to list all manual entries related to `timezone`, and save the result to `/root/tz_cmds.txt`.

**3.** Install the `httpd` package if needed, then list the contents of its documentation directory under `/usr/share/doc` and save that listing to `/root/httpd_doc.txt`.

**4.** From the `ls` man page, find the option that sorts files by **modification time**, and write just that option (e.g. `-X`) into `/root/ls_time.txt`.

**5.** Using `man`, determine the configuration file read by the `chronyd` service and write its full path into `/root/chrony_conf.txt`.
