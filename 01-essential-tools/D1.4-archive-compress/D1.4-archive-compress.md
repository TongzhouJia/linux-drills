# KP D1.4 — Archive & Compress (tar / star / gzip / bzip2 / xz)
**6 tasks · RHEL 10 · server1 as root**

**1.** Create a **gzip**-compressed tar archive of the entire `/etc` directory, named `/root/etc_backup.tar.gz`.

**2.** Create a **bzip2**-compressed tar archive of `/var/log`, named `/root/logs.tar.bz2`.

**3.** Without extracting anything, list the contents of `/root/etc_backup.tar.gz` and save that listing to `/root/etc_contents.txt`.

**4.** Extract **only** the single file `etc/hostname` out of `/root/etc_backup.tar.gz` into a new directory `/root/restore/`.

**5.** Create an **xz**-compressed tar archive of `/usr/share/doc`, named `/root/doc.tar.xz`.

**6.** Install the `star` archiver, then use it to create a **bzip2**-compressed star archive of `/etc` named `/root/etc.star.bz2`. (On RHEL 10, `star` may not be in BaseOS/AppStream — you may need to enable an extra repo to install it.)
