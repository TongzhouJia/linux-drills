# KP D1.6 — Hard & Soft Links (ln)
**6 tasks · RHEL 10 · server1 as root**

**1.** In a new directory `/root/linkdir`, create a file `original` containing the text `hello`. Create a **hard link** named `hardcopy` and a **soft (symbolic) link** named `softcopy`, both pointing to `original`. Then confirm with `ls -li` that `original` and `hardcopy` share the same inode.

**2.** Continuing from task 1, delete `original`. Verify (and be able to state) that `hardcopy` is still readable while `softcopy` is now a broken link.

**3.** Create a symbolic link `/root/etchosts` that points to `/etc/hosts`.

**4.** Create a symbolic link `/root/logdir` that points to the directory `/var/log`.

**5.** Given a file `/root/data.txt` (create it with any content), create a **hard link** `/root/data_backup` and confirm its link count is now **2**.

**6.** Create both a hard link and a soft link to any file inside `/root/linkdir`, then save the output of `ls -li /root/linkdir` to `/root/linkinfo.txt` so the inode numbers and link types are recorded.
