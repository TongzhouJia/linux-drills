# KP D1.2 — grep & Regular Expressions
**6 tasks · RHEL 10 · server1 as root · no editor, use pipes/redirection**

**1.** From `/etc/passwd`, write to `/root/nologin_users.txt` **only the usernames** (first field) of every account whose login shell is `/sbin/nologin`.

**2.** Recursively search `/etc` for every file that contains the string `PermitRootLogin` (case-insensitive). Save the **matching file paths only** (not the lines) to `/root/rootlogin_files.txt`, ignoring permission errors.

**3.** Count how many lines under `/var/log` (recursive, ignore errors) contain the word `error` case-insensitively, and write **just the number** to `/root/errcount.txt`.

**4.** From `/etc/ssh/sshd_config`, produce `/root/sshd_active.txt` containing only the **active** configuration lines — i.e. exclude comment lines (starting with `#`) and blank lines.

**5.** From `/etc/passwd`, save to `/root/bash_users.txt` every full line whose login shell (the last field) is exactly `/bin/bash`. Use an **anchored** regular expression.

**6.** From the output of `ip addr`, save to `/root/ipv4.txt` only the lines showing **IPv4** addresses (the `inet ` lines) — do not include the IPv6 `inet6` lines.
