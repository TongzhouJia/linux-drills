# KP D1.10 — SSH Access & Secure File Transfer (ssh / scp / rsync)
**6 tasks · RHEL 10 · two servers (server1 192.168.55.71, server2 192.168.55.72), root**

**1.** On server1, ensure the `sshd` service is running **and** enabled to start at boot. Confirm its status.

**2.** Copy `/root/etc_backup.tar.gz` from server1 to server2's `/tmp/` directory using `scp`.

**3.** Use `rsync` in **archive mode** to mirror `/etc` from server1 into `/root/etc_from_s1/` on server2, preserving ownership, permissions and timestamps.

**4.** Configure **key-based** SSH authentication so that `root` on server1 can `ssh` to server2 **without a password** (generate a key pair with no passphrase and install the public key on server2).

**5.** Securely copy the repo file `/etc/yum.repos.d/local.repo` from server1 to the **same path** on server2.

**6.** From server1, run `hostname` on server2 over SSH as a **single non-interactive command** and save its output to `/root/s2_hostname.txt`.
