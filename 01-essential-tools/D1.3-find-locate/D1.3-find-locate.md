# KP D1.3 — find & locate
**6 tasks · RHEL 10 · server1 as root**

**1.** Find every regular file under `/etc` larger than **1 MB** and copy each of them into a new directory `/root/bigconf/`.

**2.** Find all files under `/var` owned by the user `chrony` and save their full paths to `/root/chrony_files.txt` (ignore permission errors).

**3.** Find every file under `/home` that was **modified in the last 24 hours** and save the list to `/root/recent_home.txt`.

**4.** Find all **directories** named `scripts` anywhere under `/usr` and save their paths to `/root/scriptdirs.txt`.

**5.** Find every **empty** regular file under `/tmp` and delete them all in a single `find` command.

**6.** Using `locate`, produce `/root/conf_files.txt` listing all files whose names end in `.conf` under `/etc`. (Remember the database may need updating first, and the `mlocate`/`plocate` package may need installing.)
