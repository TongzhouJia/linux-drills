# KP D1.7 — Standard Permissions (ugo/rwx, chmod)
**6 tasks · RHEL 10 · server1 as root**

**1.** Create `/root/perms/a.txt` and set its permissions to exactly `rw-r-----` using **numeric** mode.

**2.** Create `/root/perms/script.sh` and, using **symbolic** mode, set it so the owner has read+write+execute, the group has read+execute, and others have nothing.

**3.** Recursively set everything under `/root/perms` (the directory, subdirectories, and files) to mode `750`.

**4.** File `/root/perms/b.txt` currently has mode `644`. In a **single symbolic** `chmod` command, add execute for the owner and remove read for others.

**5.** Create a directory `/root/team` that group members can enter and list, but that "others" cannot access at all (i.e. `750`).

**6.** Set `/root/perms/a.txt` so only its owner can read and write it (no group/other access), then save `ls -l /root/perms` to `/root/perms_check.txt`.
