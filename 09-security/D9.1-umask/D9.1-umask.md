# KP D9.1 — Default File Permissions (umask)
**5 tasks · RHEL 10 · server1 as root**

**1.** Configure the system so that new files are created with **660** permissions by default (i.e. set the appropriate umask), and make it apply on **both** servers.

**2.** Show the current umask value and save it to `/root/umask.txt`.

**3.** Set a umask of **077** for the root user **persistently** (in the shell startup file).

**4.** After setting umask `027`, create a file and a directory and confirm the resulting modes are `640` and `750`.

**5.** Make a umask setting apply **system-wide to all users** (via `/etc/profile.d/` or the system bashrc).
