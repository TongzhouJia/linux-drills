# KP D9.2 — Special Permissions (setuid / setgid / sticky)
**6 tasks · RHEL 10 · server1 as root**

**1.** Set the **setuid** bit on a program file and verify it shows `s` in the owner field of `ls -l`.

**2.** Set the **setgid** bit on a shared directory so that new files created inside inherit the directory's group.

**3.** Set the **sticky** bit on a world-writable directory so users can only delete their own files.

**4.** Find **all files on the system** that have the **setuid** bit set, and save the list to `/root/suid.txt`.

**5.** Using a **single numeric** `chmod`, set both **setgid and sticky** on a directory (mode `3770`).

**6.** **Remove** the setuid bit from a file that currently has it.
