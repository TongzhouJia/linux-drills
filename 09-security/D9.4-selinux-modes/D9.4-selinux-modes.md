# KP D9.4 — SELinux Modes (enforcing / permissive)
**5 tasks · RHEL 10 · server2 as root**

**1.** Put SELinux on server2 into **permissive** mode **temporarily** (effective immediately, no reboot).

**2.** Make the SELinux mode change **permanent** across reboots (edit the config file).

**3.** Show the current SELinux mode and full status, and save it to `/root/selinux_status.txt`.

**4.** Switch SELinux back to **enforcing** **without** rebooting.

**5.** Determine, from the config file alone, which mode SELinux will be in **after the next reboot**, and write the answer to `/root/next_boot_mode.txt`.
