**3.** Restore the **default** SELinux context on a file whose context was changed.

**6.** Add a file-context **rule** so an entire directory tree keeps the correct type even after a relabel, then relabel it **recursively**.

**7.** A web server serves files that are inaccessible. **Without changing any web server configuration files**, make all the files accessible (this is a pure SELinux context fix). Verify it works with SELinux enforcing.
