# KP D9.3 — Key-based SSH Authentication
**5 tasks · RHEL 10 · server1 (.71) & server2 (.72), root**

**1.** Enable SSH access for **root** on both servers (ensure sshd allows root login and is running/enabled).

**2.** Set up **key-based** authentication so that root on **server1** can `ssh` to **server2** **without a password**.

**3.** Generate an **ed25519** key pair with **no passphrase**.

**4.** Copy a public key to a remote user's `authorized_keys` (using `ssh-copy-id`).

**5.** Harden sshd by **disabling password authentication** (key-only) in `sshd_config` and reload the service.
