# KP D4.3 — Logical Volumes: Create & Delete (lvcreate / lvremove)
**6 tasks · RHEL 10 · server2 as root · use `data_vg` from D4.2**

**1.** In `data_vg`, create a **500 MiB** logical volume named `data_lv`.

**2.** Create an LV named `app_lv` sized by **extents** — use exactly **50 extents** (`-l`).

**3.** Create an LV named `full_lv` that consumes **100% of the free space** in a given VG.

**4.** Format `data_lv` as **xfs** and mount it at `/mnt/data_lv` (temporary mount is fine for this task).

**5.** Display all logical volumes and save the output to `/root/lvs.txt`.

**6.** Delete the logical volume `app_lv` cleanly (unmount first if it is mounted).
