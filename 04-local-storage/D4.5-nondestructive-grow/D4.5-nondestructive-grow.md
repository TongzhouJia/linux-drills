# KP D4.5 — Add Partitions / LVs / Swap Non-destructively
**7 tasks · RHEL 10 · ⭐⭐⭐ server2 as root · existing data must NOT be lost**

**1.** `/mnt/data_lv` (xfs) is full. Extend the underlying LV by **500 MiB** and grow the xfs filesystem **online** — no unmount, no data loss.

**2.** Add a brand-new disk `/dev/sdd`, create a PV on it, and **extend** an existing VG with it — without disturbing existing LVs.

**3.** On a disk that already holds data partitions, add a **new 500 MiB swap partition** without touching the others, and enable it **persistently**.

**4.** Extend an **ext4** logical volume and grow its filesystem with `resize2fs`.

**5.** Add a **new logical volume** to a VG that already hosts other LVs (leave them untouched), format it and mount it persistently.

**6.** Extend an LV using **all remaining free extents** in the VG (`+100%FREE`) and grow the filesystem.

**7.** After all the non-destructive changes, verify the original data is intact and the new sizes show correctly in `df -h`; save `df -h` to `/root/df_after.txt`.
