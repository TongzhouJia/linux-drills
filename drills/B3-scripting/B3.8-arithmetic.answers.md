# KP B3.8 答案 — Arithmetic & Numbers

> 全部真机跑过、输出核对无误。心法：**整数用 `$(( ))`，小数用 `bc` 或 `printf`。**

---

**1.** 整数运算 `$(( ))`。
```bash
echo $((3+4*2))     # → 11
```
*为什么：* `$(( ))` 做整数运算，遵守优先级（先乘后加）。里面用变量可不加 `$`：`$((a+b))`。

**2.** 自增。
```bash
n=5; ((n++)); echo $n     # → 6
```
*为什么：* `(( ))` 是"算术命令"，专门做赋值/自增。`((n++))`、`((n+=3))`、`((n*=2))` 都行。（注意 `((n++))` 当 n=0 时返回退出码 1，在 `set -e` 下要小心，用 `((n++)) || true` 或 `: $((n++))`。）

**3.** 三元表达式。
```bash
a=7; echo $(( a>5 ? 100 : 0 ))    # → 100
```
*为什么：* `$(( 条件 ? 真值 : 假值 ))` 和 C 一样。快速二选一取值。

**4.** 整除与取模。
```bash
echo "$((17/5)) $((17%5))"    # → 3 2
```
*为什么：* Bash 除法是**整数除**（截断，不四舍五入），`%` 取余。判断奇偶、每 N 个一批、循环取模都靠它。

**5.** `bc` 做浮点。
```bash
echo "scale=2; 10/3" | bc     # → 3.33
```
*为什么：* Bash 不做小数，`bc` 可以。`scale=2` 设小数位数。也可 `bc -l`（带数学库，支持 `s()`、`sqrt()` 等）。注意 `bc` 从 stdin 读表达式。

**6.** `printf` 取整/格式化。
```bash
printf "%.2f\n" 3.14159       # → 3.14
```
*为什么：* `printf "%.2f"` 四舍五入到 2 位（和 `bc` 的截断不同）。出报表、算占比时用它控制小数位最方便。

**7.** 进制转换。
```bash
echo "$((16#FF)) $((2#1010))"   # → 255 10   （十六进制/二进制 → 十进制）
printf "%x\n" 255               # → ff        （十进制 → 十六进制）
```
*为什么：* `$((base#num))` 把任意进制读成十进制（`16#` 十六进制、`2#` 二进制、`8#` 八进制）。反向输出用 `printf`：`%x` 十六进制、`%o` 八进制。看权限位、颜色码、掩码常用。

**8.** `$RANDOM` 随机数。
```bash
echo $((RANDOM%6+1))          # → 1..6 的随机整数（掷骰子）
```
*为什么：* `$RANDOM` 每次给 0–32767 的随机数，`%6+1` 映射到 1–6。生成临时后缀、随机延迟（`sleep $((RANDOM%5))`）打散定时任务负载很实用。

---

## 一句话总结（B3.8 核心）
- **整数**：`$(( ))` 运算、`(( ))` 赋值/自增/比较；`/` 是整除、`%` 取余；三元 `?:`。
- **小数**：`bc`（`scale=N`，从 stdin 读）或 `printf "%.2f"`（会四舍五入）。
- **进制**：`$((16#FF))` 读入、`printf "%x"` 输出；`$RANDOM` 出随机数。
