# KP B3.4 答案 — Functions

> 全部真机跑过、输出核对无误。心法：**函数内变量用 `local`；"返回数据"靠 `echo`+`$(func)`，`return` 只表示成败(0–255)。**

---

**1.** 定义与调用。
```bash
greet(){ echo "Hi, $1"; }
greet Tongzhou        # → Hi, Tongzhou
```
*为什么：* `name(){ ...; }` 定义，`name args` 调用。函数参数用 `$1 $2 …`，和脚本参数一样。

**2.** `local` 作用域。
```bash
f(){ local x=inside; echo "in:$x"; }
x=outside; f; echo "out:$x"    # → in:inside / out:outside
```
*为什么：* bash 变量**默认全局**——函数里改会污染外面。`local x` 把 `x` 限定在函数内。**函数里所有临时变量都该 `local`**，否则大脚本里神秘串值。

**3.** 用返回状态做判断。
```bash
is_even(){ (( $1 % 2 == 0 )); }
if is_even 4; then echo "4 even"; fi     # → 4 even
is_even 5 && echo even || echo "5 odd"   # → 5 odd
```
*为什么：* 函数最后一条命令的退出码就是函数退出码（这里 `(( ))` 偶数为真=0）。所以函数能直接放进 `if`/`&&`/`||`。`return N` 可显式指定(0=成功)。

**4.** 捕获输出。
```bash
sum(){ echo $(( $1 + $2 )); }
r=$(sum 3 4); echo "r=$r"      # → r=7
```
*为什么：* bash 函数**不能返回字符串/数字**给调用者，只能 `echo` 出来、用 `$(func)` 捕获。这是最常见的"函数当计算器"模式。

**5.** 参数个数与列表。
```bash
count_args(){ echo "n=$#: $*"; }
count_args a b c             # → n=3: a b c
```
*为什么：* 函数内 `$#`=参数个数、`$1..$n`=各参数、`"$@"`=全部（**推荐**，能正确保留带空格的参数）、`$*`=全部拼成一串。

**6.** 默认参数。
```bash
greet2(){ local name=${1:-guest}; echo "Welcome, $name"; }
greet2;       # → Welcome, guest
greet2 Alice  # → Welcome, Alice
```
*为什么：* `local name=${1:-guest}` = 没给第一个参数就用 `guest`（B1.3 的参数展开）。给函数配"可选参数+默认值"很方便。

**7.** 用 `echo` 返回字符串。
```bash
maxof(){ if (( $1 > $2 )); then echo "$1"; else echo "$2"; fi; }
echo "max=$(maxof 7 3)"      # → max=7
```
*为什么：* 需要"返回"较大值这种数据，同样靠 `echo` + 捕获。别指望 `return` 传值（它只有 0–255，且是退出码语义）。

**8.** 函数进循环（组合）。
```bash
double(){ echo $(( $1 * 2 )); }
for n in 1 2 3; do printf '%s ' "$(double "$n")"; done; echo   # → 2 4 6
```
*为什么：* 把重复逻辑封成函数，循环里反复调用，脚本更干净。注意调用时给参数加引号 `"$n"`。

---

## 一句话总结（B3.4 核心）
- 定义 `name(){ …; }`，参数 `$1 $#  "$@"`（`"$@"` 最稳）。
- **函数内变量必 `local`**，否则污染全局。
- **返回数据 = `echo` + `$(func)` 捕获**；`return N` 只表示成败(0–255)，能直接用于 `if`。
- 默认参数 `local x=${1:-默认}`。
