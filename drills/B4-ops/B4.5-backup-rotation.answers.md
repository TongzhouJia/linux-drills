# KP B4.5 答案 — Backup & Rotation

> 全部真机跑过、输出核对无误。心法：**`tar` 冷备打包、`rsync` 增量同步；备份脚本 = 日期命名 + 保留 N 份删旧。**

---

**1–4. tar 打包/查看/解包。**
```bash
tar czf backup.tar.gz src          # c 建 z gzip f 文件；打包 src/
tar czf "backup-$(date +%F).tar.gz" src   # 按日期命名
tar tzf backup.tar.gz              # t 列出内容（不解包）
tar xzf backup.tar.gz -C restore   # x 解包，-C 指定目标目录
```
*为什么：* 记 `c`reate / `t`list / e`x`tract 三个动作 + `z`(gzip) `f`(file)。**先 `tzf` 看一眼再 `xzf`**，避免解出一堆东西污染当前目录；`-C` 指定解到哪。

**5–7. rsync 同步。**
```bash
rsync -a src/ mirror/                    # -a 归档模式（保留权限/时间/软链）
rsync -a --delete src/ mirror/           # 目标多出来的文件也删，严格镜像
rsync -a --exclude='*.tmp' src/ clean/   # 排除临时文件
```
*为什么：* `rsync` 只传**变化的部分**（增量），比反复全量 `cp` 快得多。
- ⚠️ **尾部斜杠**：`src/`（带斜杠）= 复制"src 里的内容"；`src`（不带）= 复制"src 这个目录本身"。差一个斜杠结果不同，务必留意。
- `--delete` 让目标和源**完全一致**（危险，会删目标多余文件，先 `--dry-run` 看看）。

**8.** 保留最近 N 份。
```bash
ls -1t backup-*.tar.gz | tail -n +4 | xargs -r rm -f
```
*为什么：* `ls -1t` 按时间**新→旧**排；`tail -n +4` = 从第 4 个起（即跳过最新 3 个）；`xargs -r rm -f` 删掉其余。`-r`（--no-run-if-empty）保证没有多余文件时不空跑 `rm`。这就是"日志/备份轮转"的核心。

**9.** 备份脚本（日期 + 保留）。
```bash
#!/usr/bin/env bash
set -euo pipefail
SRC=${1:?src}; DEST=${2:?dest}; KEEP=${3:-3}
mkdir -p "$DEST"
ts=$(date +%F_%H%M%S)
archive="$DEST/backup-$ts.tar.gz"
tar czf "$archive" -C "$(dirname "$SRC")" "$(basename "$SRC")"
echo "created $archive"
ls -1t "$DEST"/backup-*.tar.gz | tail -n +$((KEEP+1)) | xargs -r rm -f
echo "kept $(ls -1 "$DEST"/backup-*.tar.gz | wc -l) archives"
```
*为什么：* 时间戳含时分秒（`%F_%H%M%S`）避免同日覆盖；`tar -C "$(dirname)" "$(basename)"` 让归档里是**相对路径**（不带一长串 `/tmp/...`，解包更干净）；保留数做成参数。挂到 cron/timer（B5.3）就是自动备份。

**10.** 校验备份。
```bash
find src -type f | wc -l                       # 源文件数
tar tzf backup.tar.gz | grep -c '[^/]$'        # 归档里的文件数（排除目录行）
# 两者相等 → 3 = 3
```
*为什么：* **备份完要校验**，否则"以为备了其实是空的"。简单法比文件数；严格法解包后 `diff -r` 或比对校验和。`grep -c '[^/]$'` 数不以 `/` 结尾的行（即文件，非目录）。

---

## 一句话总结（B4.5 核心）
- **tar**：`czf` 打包、`tzf` 看、`xzf -C dir` 解；配 `-$(date +%F)` 命名、`-C` 控制路径。
- **rsync**：`-a` 归档增量同步，`--delete` 严格镜像（危险先 dry-run），`--exclude` 排除；**注意尾斜杠**。
- **轮转**：`ls -1t | tail -n +N+1 | xargs -r rm -f` 只留最近 N 份。
- 备份脚本 = 时间戳命名 + 轮转 + 校验，挂 cron/timer 自动跑。
