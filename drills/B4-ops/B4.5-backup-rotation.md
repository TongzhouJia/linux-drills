# KP B4.5 — Backup & Rotation
**`tar` 打包 · 按日期命名 · `rsync -a`/`--delete`/`--exclude` · 保留 N 份删旧 · 备份脚本 · 校验**

**10 tasks · ⭐⭐ · RHEL 10 · server1 · run each and match the result**

> 备份是运维基本功。`tar` 打冷备份、`rsync` 做增量同步/镜像；再加"按日期命名 + 只留最近 N 份"就是能上生产的备份脚本。先建个 `src/` 练手。

**Setup：** `mkdir -p src/{a,b}; echo hi>src/a/f1.txt; echo yo>src/b/f2.txt; echo t>src/scratch.tmp`

---

**1.** Create a gzip tarball of `src/`.
*Acceptance:* `tar czf backup.tar.gz src` → archive created.

**2.** Create a **date-stamped** archive name.
*Acceptance:* `tar czf "backup-$(date +%F).tar.gz" src` → e.g. `backup-2026-07-18.tar.gz`.

**3.** List an archive's contents **without extracting**.
*Acceptance:* `tar tzf backup.tar.gz` → the file tree inside.

**4.** Extract an archive into a specific directory.
*Acceptance:* `tar xzf backup.tar.gz -C restore` → files appear under `restore/`.

**5.** Mirror `src/` to `mirror/` with `rsync -a` (preserve perms/times).
*Acceptance:* `rsync -a src/ mirror/` → `mirror/` matches `src/`.

**6.** Make the mirror **exactly** match the source (delete extras) with `--delete`.
*Acceptance:* a stale file in `mirror/` is removed after `rsync -a --delete src/ mirror/`.

**7.** Sync but **exclude** `*.tmp` files.
*Acceptance:* `rsync -a --exclude='*.tmp' src/ clean/` → `clean/scratch.tmp` does **not** exist.

**8.** **Retention**: given several `backup-*.tar.gz`, keep the 3 newest and delete older.
*Acceptance:* `ls -1t backup-*.tar.gz | tail -n +4 | xargs -r rm -f` → only the 3 most-recent remain.

**9.** Write `backup.sh <src> <dest> [keep]` that makes a timestamped tarball into `<dest>` and prunes to the newest `keep` archives.
*Acceptance:* creates `backup-<timestamp>.tar.gz` and reports how many archives are kept.

**10.** **Verify** a backup: compare the file count in the archive to the source.
*Acceptance:* `find src -type f | wc -l` equals `tar tzf backup.tar.gz | grep -c '[^/]$'` (both `3`).
