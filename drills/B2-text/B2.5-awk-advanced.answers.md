# KP B2.5 答案 — awk Advanced

> 全部真机跑过、输出核对无误。心法：**`BEGIN{}` 开场一次 → 逐行累加 → `END{}` 收尾一次**。变量不用声明，默认 0/空。

---

**1.** 求和。
```bash
awk '{s+=$NF} END{print s}' access.log      # → 23936
```
*为什么：* `s+=$NF` 每行把最后一列累加到 `s`（初值自动为 0），`END` 里打印总和。日志总流量、总请求数都这么算。

**2.** 求平均。
```bash
awk '{s+=$NF; n++} END{printf "%.1f\n", s/n}' access.log   # → 797.9
```
*为什么：* 同时累加总和 `s` 和计数 `n`，`END` 里相除。`printf "%.1f"` 控制小数位。

**3.** 计算字段求和（营收）。
```bash
awk -F, 'NR>1{r+=$4*$5} END{printf "%.2f\n", r}' sales.csv   # → 506.31
```
*为什么：* awk 能就地算术 `$4*$5`（数量×单价）再累加。`NR>1` 跳表头。一条命令算完整张表的营收。

**4.** 正则匹配整行 `~`。
```bash
awk '$0 ~ /\/api\// {c++} END{print c}' access.log   # → 10
```
*为什么：* `$0 ~ /正则/` = 整行匹配就计数（`/` 要转义成 `\/`）。也可对某列匹配：`$7 ~ /api/`。`!~` 是不匹配。

**5.** `BEGIN` 打表头。
```bash
awk -F, 'BEGIN{print "PRODUCT UNITS"} NR>1{print $3,$4}' sales.csv
```
*为什么：* `BEGIN{}` 在读第一行**之前**执行一次，适合打表头、初始化变量、设 `FS`/`OFS`。

**6.** `if/else` 分类计数。
```bash
awk '{if($9>=500)e++; else if($9>=400)w++; else ok++} END{print "ok="ok,"warn="w,"err="e}' access.log
# → ok=19 warn=6 err=5
```
*为什么：* action 里可写完整 `if/else if/else`。一遍扫描把请求分成 2xx/3xx、4xx、5xx 三档。运维健康报表的雏形。

**7.** 逐行算 + 对齐。
```bash
awk -F, 'NR>1{printf "%-8s %.2f\n", $3, $4*$5}' sales.csv
# → Widget   25.00 / Gadget   49.95 / Gizmo    42.00 ...
```
*为什么：* 边算边用 `printf` 出对齐明细。做"每项营收清单"直接成型。

**8.** 在 `BEGIN` 设 `FS`/`OFS`。
```bash
awk 'BEGIN{FS=":"; OFS=" | "} {print $1,$3}' passwd.sample   # → root | 0 ...
```
*为什么：* 除了命令行 `-F`，也能在 `BEGIN` 里设 `FS`（输入分隔符）和 `OFS`（输出分隔符）。`print $1,$3` 的逗号会用 `OFS` 连接，一步换分隔符。

**9.** 模式块计数。
```bash
awk '/Failed password/{c++} END{print c}' syslog.log   # → 4
```
*为什么：* `/正则/{action}` 是 awk 最简写法（等价 `$0 ~ /.../`）。统计某类事件次数。

**10.** 追踪最大值。
```bash
awk '$NF>max{max=$NF} END{print max}' access.log   # → 4096
```
*为什么：* `$NF>max` 为真才更新 `max`（初值 0），`END` 打印。求最大响应、最长耗时、峰值都用这招；求最小值把 `>` 改 `<` 并初始化一个大数。

---

## 一句话总结（B2.5 核心）
- **三段结构**：`BEGIN{}` 初始化/表头 → `{每行累加}` → `END{}` 出结果。
- **算账**：`s+=$NF` 求和、`s/n` 平均、`$4*$5` 计算字段；`printf "%.2f"` 控精度。
- **判断**：`if/else if/else` 分类计数；正则 `$0 ~ /pat/`、`/pat/{...}`；max/min 用"比就更新"。
- `FS`/`OFS` 可在 `BEGIN` 设。这套就是"扫一遍日志出一份指标"的通用骨架。
