# KP B2.3 答案 — sed Advanced

> 全部真机跑过、输出核对无误。心法：**`\( \)` 分组、`\1 \2` 回引**（用 `-E` 时写 `( )`）；地址可以是"从模式到模式"。

---

**1.** 一次多条命令 `-e`。
```bash
sed -e '1d' -e 's/,/\t/g' sales.csv | head -1 | cat -A
# → 2026-07-01^IEast^IWidget^I10^I2.50$   （^I 就是 Tab）
```
*为什么：* 每个 `-e` 挂一条命令，按顺序在**同一遍**里执行（先删表头、再换分隔符）。也可写成 `sed '1d; s/,/\t/g'`。`cat -A` 用来看不可见的 Tab/行尾。

**2.** 分组回引重排字段。
```bash
sed -E 's/^([^:]+):.*:([^:]+)$/\1 -> \2/' passwd.sample
# → root -> /bin/bash   daemon -> /usr/sbin/nologin   ...
```
*为什么：* `([^:]+)` 抓第一段（用户名），`([^:]+)$` 抓最后一段（shell），`\1 \2` 在替换里复用。`-E` 让 `( )` 直接生效（否则要写 `\( \)`）。

**3.** 三组回引，日期换格式。
```bash
echo "2026-07-01" | sed -E 's#([0-9]+)-([0-9]+)-([0-9]+)#\3/\2/\1#'   # → 01/07/2026
```
*为什么：* 三个分组分别是年月日，替换里按 `\3/\2/\1` 重新拼。日志时间戳、文件名日期改格式全靠这招。

**4.** 范围地址"从模式到模式"。
```bash
sed -n '/11:00:00/,/11:03:00/p' access.log
# → 从第一条 11:00:00 到第一条 11:03:00 之间的 4 行
```
*为什么：* 地址可写成 `/起/,/止/`，打印**一段区间**。抓"从某事件到某事件"的日志片段极常用。

**5.** `y///` 转写（按位一一映射）。
```bash
echo "abc" | sed 'y/abc/ABC/'      # → ABC
```
*为什么：* `y/源/目标/` 按字符逐一替换（等价 `tr abc ABC`），常用于大小写或字符集转换。注意它不是正则，是逐字符对应。

**6.** 地址 + 替换（只改符合条件的行）。
```bash
sed '/East/ s/Widget/WIDGET/' sales.csv
# → 只有含 East 的行里的 Widget 变 WIDGET；North/West 的 Widget 不动
```
*为什么：* 在 `s` 前面加地址 `/East/`，替换**只作用于匹配行**。"仅对某类记录做修改"的标准写法。

**7.** 删空行。
```bash
printf 'a\n\nb\n\n\nc\n' | sed '/^$/d'   # → a b c
```
*为什么：* `/^$/` 匹配"空行"（行首紧接行尾），`d` 删除。清洗数据去空行常用；压缩连续空行也可用 `cat -s`。

**8.** 追加行 `a`。
```bash
printf 'a\nb\nc\n' | sed '2a INSERTED'   # → a / b / INSERTED / c
```
*为什么：* `2a text` 在第 2 行**后**追加一行；`i` 是**前**插入，`c` 是整行替换。给文件加注释行/分隔行时用。（GNU sed 支持 `a text` 一行写法；老式可移植写法是 `a\` 换行再写内容。）

---

## 一句话总结（B2.3 核心）
- **多步一遍过**：`-e cmd1 -e cmd2` 或 `'cmd1; cmd2'`。
- **分组回引**：`\( \)`/`( )`+`-E` 抓片段，`\1 \2` 在替换里重排——改日期、换字段顺序的利器。
- **范围地址** `/起/,/止/` 抓区间；**地址+s** 只改匹配行；`y///` 逐字符转写；`a/i/c/d` 增删改整行。
