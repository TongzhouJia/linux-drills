# KP B2.4 — awk Basics
**字段 `$1..$NF` · `$0` · `NR`/`NF` · `-F` 分隔符 · `print` / `printf` · `pattern{action}`**

**10 tasks · ⭐⭐ · RHEL 10 · server1 · run each against the sample files and match the result**

> 样例在 `samples/`。awk 是三剑客里**最值钱**的一个：它把每行自动切成字段 `$1 $2 …`，天生用来"取列、筛选、算数、出报表"。这一节先把字段和过滤玩熟。
> 记忆：`$0`=整行，`$1`=第一列，`$NF`=最后一列，`NR`=当前行号，`NF`=本行列数，`-F` 设分隔符。

---

**1.** Print just the usernames (field 1) from `passwd.sample`.
*Acceptance:* `awk -F: '{print $1}'` → `root daemon bin tongzhou alice bob carol svc_web svc_db dave`.

**2.** Print each user with their shell (fields 1 and 7).
*Acceptance:* `awk -F: '{print $1, $7}'` → `root /bin/bash`, `daemon /usr/sbin/nologin`, … (comma = space via OFS).

**3.** Print the **last** field (bytes) of each `access.log` line, and confirm every line has the same field count.
*Acceptance:* `awk '{print $NF}'` prints the bytes column; `awk '{print NF}' … | sort -u` → `10`.

**4.** Print the total number of records in `access.log`.
*Acceptance:* `awk 'END{print NR}'` → `30`.

**5.** From `sales.csv` (comma-separated), print product and units, **skipping the header**.
*Acceptance:* `awk -F, 'NR>1{print $3, $4}'` → `Widget 10`, `Gadget 5`, `Gizmo 3`, …

**6.** Count how many `access.log` requests had an HTTP status (field 9) `>= 400`.
*Acceptance:* `awk '$9 >= 400' access.log | wc -l` → `11`.

**7.** Print the usernames whose login shell (field 7) is exactly `/bin/bash`.
*Acceptance:* `awk -F: '$7=="/bin/bash"{print $1}'` → `root tongzhou alice dave`.

**8.** Using `printf`, print each product left-aligned in 8 chars and its units right-aligned in 3 (header skipped).
*Acceptance:* `awk -F, 'NR>1{printf "%-8s %3d\n", $3, $4}'` → `Widget    10`, `Gadget     5`, …

**9.** The status code is the **second-to-last** field. Print it as `$(NF-1)` and tally the codes.
*Acceptance:* `awk '{print $(NF-1)}' access.log | sort | uniq -c` → `17 200`, `2 301`, `1 401`, `5 404`, `3 500`, `2 503`.

**10.** Print a sentence per line combining fields with literal text.
*Acceptance:* `awk -F: '{print $1 " uses " $7}'` → `root uses /bin/bash`, …
