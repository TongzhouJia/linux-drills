# KP B2.4 答案 — awk Basics

> 全部真机跑过、输出核对无误。心法：**awk 自动按分隔符切列**，`pattern{action}` = "符合 pattern 的行就做 action"，省了大量 `if`。

---

**1.** 取某列。
```bash
awk -F: '{print $1}' passwd.sample
# → root daemon bin tongzhou alice bob carol svc_web svc_db dave
```
*为什么：* `-F:` 把每行按 `:` 切开，`$1` 就是第一列（用户名）。等价 `cut -d: -f1`，但 awk 能顺带筛选/算数。

**2.** 取多列（逗号=空格 OFS）。
```bash
awk -F: '{print $1, $7}' passwd.sample     # → root /bin/bash ...
```
*为什么：* `print` 里用**逗号**分隔各项，输出时会插入 `OFS`（默认空格）。若写 `print $1 $7`（无逗号）会**直接拼接**成 `root/bin/bash`——这是新手常见坑。

**3.** 最后一列 `$NF` 与列数 `NF`。
```bash
awk '{print $NF}' access.log        # 每行最后一列（字节数）
awk '{print NF}' access.log | sort -u   # → 10（每行都 10 列）
```
*为什么：* `NF` = 本行字段数，`$NF` = 最后一个字段（不用数到第几列）。日志列数固定时，`$NF`/`$(NF-1)` 定位末尾字段最稳。

**4.** 记录数 `NR` + `END`。
```bash
awk 'END{print NR}' access.log      # → 30
```
*为什么：* `NR` 是"到目前为止的行号"，处理完最后一行时 `NR` = 总行数；`END{}` 块在读完所有行后执行一次。等价 `wc -l`。

**5.** CSV 跳表头。
```bash
awk -F, 'NR>1{print $3, $4}' sales.csv    # → Widget 10 / Gadget 5 / ...
```
*为什么：* `NR>1` 作为 pattern，跳过第 1 行（表头），只对数据行执行。处理 CSV 的标配。

**6.** 数值条件过滤。
```bash
awk '$9 >= 400' access.log | wc -l   # → 11
```
*为什么：* `$9 >= 400` 是 pattern，为真则默认动作（打印整行）。awk 里字段能当**数字**直接比较。这里筛出所有 4xx/5xx 错误请求（共 11 条）。

**7.** 字符串条件过滤。
```bash
awk -F: '$7=="/bin/bash"{print $1}' passwd.sample   # → root tongzhou alice dave
```
*为什么：* 字符串相等用 `==` 加引号。"某列等于某值就取另一列"——比 `grep`+`cut` 组合更精确（不会误匹配子串）。

**8.** `printf` 对齐输出。
```bash
awk -F, 'NR>1{printf "%-8s %3d\n", $3, $4}' sales.csv
# → Widget    10
#   Gadget     5
```
*为什么：* `printf` 要**自己写 `\n`**。`%-8s`=左对齐 8 字符串，`%3d`=右对齐 3 位整数。出对齐报表必用（`print` 做不到列对齐）。

**9.** 倒数第二列 `$(NF-1)`。
```bash
awk '{print $(NF-1)}' access.log | sort | uniq -c
# → 17 200 · 2 301 · 1 401 · 5 404 · 3 500 · 2 503
```
*为什么：* 字段号可以是**表达式** `$(NF-1)`（倒数第二列=状态码）。配 `sort | uniq -c` 就是"各状态码计数"的经典管道（B2.6 会用 awk 数组一步搞定）。

**10.** 字段与文本拼接。
```bash
awk -F: '{print $1 " uses " $7}' passwd.sample   # → root uses /bin/bash ...
```
*为什么：* 字符串**相邻即拼接**（中间放字面 `" uses "`）。想要空格就自己写进字符串，或用逗号靠 OFS。

---

## 一句话总结（B2.4 核心）
- **切列**：`-F` 设分隔符，`$1..$NF` 取字段，`$0` 整行；`print $1,$7`（逗号=OFS 空格），无逗号=硬拼接。
- **内建变量**：`NR` 行号/总数、`NF` 列数、`$(NF-1)` 倒数第二列。
- **过滤**：`pattern{action}`，数值 `$9>=400`、字符串 `$7=="..."`；`NR>1` 跳表头；`END{}` 收尾统计。
- **对齐输出**用 `printf`（记得自己写 `\n`）。
