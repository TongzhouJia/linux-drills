# KP B2.7 答案 — Field & Set Tools

> 全部真机跑过、输出核对无误。心法：**简单取列/排序/去重/转换，这些小工具比 awk 更快手；复杂逻辑再上 awk。**

---

**1.** `cut` 取列。
```bash
cut -d: -f1 passwd.sample        # → root daemon bin ...
```
*为什么：* `-d:` 设分隔符、`-f1` 取第 1 列。轻量取列首选；但 `cut` 不能处理"多个空格当一个分隔符"，那种情况用 awk。

**2.** `cut` 取多列。
```bash
cut -d: -f1,7 passwd.sample      # → root:/bin/bash ...
```
*为什么：* `-f1,7` 取第 1 和第 7 列，输出仍用原分隔符 `:` 连接。也可用范围 `-f1-3`。

**3.** `sort` 按键排序。
```bash
sort -t: -k3 -n passwd.sample    # 按 UID(第3列)数值升序
```
*为什么：* `-t:` 设字段分隔符、`-k3` 指定第 3 列为排序键、`-n` 数值排序（否则 "1000" < "2" 会按字典序排错）。降序加 `-r`。

**4.** `sort | uniq -c` 计数（黄金搭档）。
```bash
awk '{print $9}' access.log | sort | uniq -c
# → 17 200 / 2 301 / 1 401 / 5 404 / 3 500 / 2 503
```
*为什么：* `uniq -c` 统计**相邻**重复的次数，所以**必须先 `sort`** 把相同项排到一起。这是不写 awk 数组也能做分组计数的经典管道。

**5.** top-1。
```bash
awk '{print $1}' access.log | sort | uniq -c | sort -rn | head -1   # → 10 10.0.0.1
```
*为什么：* `sort | uniq -c | sort -rn | head` 四连 = "谁最多"。和 B2.6 的 awk 数组殊途同归，短平快。

**6.** `tr` 转换字符。
```bash
echo "hello world" | tr 'a-z' 'A-Z'    # → HELLO WORLD
```
*为什么：* `tr 集合1 集合2` 按位置一一映射（大小写、ROT 等）。注意 `tr` 只吃 stdin，不接文件名参数。

**7.** `tr -d` 删除、`tr -s` 压缩。
```bash
echo "a1b2c3" | tr -d '0-9'      # → abc     删掉所有数字
echo "a   b     c" | tr -s ' '   # → a b c   连续空格压成一个
```
*为什么：* `-d` 删除属于集合的字符；`-s` 把连续重复字符"挤"成一个。清洗脏数据（去多余空格、删控制字符 `tr -d '\r'` 去 Windows 换行）常用。

**8.** `paste` 并排拼接。
```bash
paste <(cut -d: -f1 passwd.sample) <(cut -d: -f7 passwd.sample)
# → root<Tab>/bin/bash ...
```
*为什么：* `paste` 把多个输入**按行并排**合并（默认 Tab 分隔）。`<(...)` 进程替换把命令输出当文件喂给它。与 `cat`(纵向接) 相对，`paste` 是横向拼。

**9.** `join` 按键连接两份数据。
```bash
join <(awk -F: '{print $3,$1}' passwd.sample | sort) \
     <(awk -F: '{print $3,$7}' passwd.sample | sort)
# → 0 root /bin/bash / 1 daemon /usr/sbin/nologin / 1000 tongzhou /bin/bash ...
```
*为什么：* `join` 像数据库 JOIN，按**公共键**（默认第 1 列）合并两文件。**前提：两边都按连接键排好序**（这里都 `| sort`），否则漏行。跨文件关联（用户↔配额、主机↔状态）用它。

**10.** `column -t` 对齐成表。
```bash
awk '{print $9}' access.log | sort | uniq -c | column -t
# → 17  200
#   2   301  ...
```
*为什么：* `column -t` 按空白自动对齐成整齐表格，报表出屏更好看。想指定分隔符用 `column -t -s,`。

---

## 一句话总结（B2.7 核心）
- **取列** `cut -d -f`（简单场景，等宽/多空格用 awk）；**排序** `sort -t -k -n -r`（数值必加 `-n`）。
- **去重计数** `sort | uniq -c`（uniq 只看相邻，先 sort）；top-N 再接 `sort -rn | head`。
- **转换** `tr`（映射/`-d`删/`-s`压），只吃 stdin；**拼接** `paste`(横向)、`join`(按键，需先排序)；**对齐** `column -t`。
