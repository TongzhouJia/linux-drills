# KP B2.1 — grep & Regular Expressions
**fixed strings · `-i -c -v -n -w -o` · BRE/ERE `-E` · anchors `^ $` · classes · context `-A/-B`**

**10 tasks · ⭐⭐ · RHEL 10 · server1 · run each against the sample files and match the result**

> 样例数据在 `samples/`（`access.log` `syslog.log` `passwd.sample` `sales.csv`）。在 server1 上跑 `bash samples/make-samples.sh` 可一键生成。假设你已 `cd` 到 `samples/`。
> grep 心法：**默认基本正则(BRE)**，`+ ? | ( ) { }` 要么加反斜杠、要么用 `-E`（扩展正则 ERE）省事。

---

**1.** Count how many `Failed password` lines are in `syslog.log`.
*Acceptance:* `grep -c` → `4`.

**2.** Case-insensitively show the line(s) mentioning `error` in `syslog.log`.
*Acceptance:* `-i` matches the `EXT4-fs error` line regardless of case.

**3.** Count the `syslog.log` lines that are **not** from `sshd`.
*Acceptance:* `grep -v 'sshd' … | wc -l` → `8`.

**4.** Show, with line numbers, every `syslog.log` line from the `kernel`.
*Acceptance:* `-n` → lines `4` and `11`.

**5.** In `passwd.sample`, match `root` as a **whole word** only (not as part of another token).
*Acceptance:* `-w root` → just the `root:x:0:0:…` line.

**6.** Extract every IPv4 address that appears in `syslog.log`, one per line, de-duplicated.
*Acceptance:* `grep -oE '([0-9]{1,3}\.){3}[0-9]{1,3}' … | sort -u` → the 6 distinct IPs (`10.0.0.1/2/3/5`, `198.51.100.9`, `203.0.113.7`).

**7.** Count the service accounts in `passwd.sample` whose username starts with `svc` (anchor at line start).
*Acceptance:* `grep -c '^svc'` → `2`.

**8.** List the users in `passwd.sample` whose login shell is `/bin/bash` (anchor at line end).
*Acceptance:* `grep '/bin/bash$'` → `root`, `tongzhou`, `alice`, `dave`.

**9.** Using an ERE alternation, count how many `access.log` requests returned status `404` **or** `500`.
*Acceptance:* `grep -E ' (404|500) ' access.log | wc -l` → `8`.

**10.** Show the `Killed process` line in `syslog.log` **together with the one line before it** (context).
*Acceptance:* `grep -B1 'Killed process'` → the `cron … backup.sh` line then the `kernel … Killed process` line.
