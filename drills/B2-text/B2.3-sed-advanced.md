# KP B2.3 — sed Advanced
**多命令 `-e`/`;` · 分组反引用 `\1` · 范围地址 `/a/,/b/` · `y///` · 地址+替换 · 增删行 `a/i/d`**

**8 tasks · ⭐⭐ · RHEL 10 · server1 · run each against the sample files and match the result**

> 样例在 `samples/`。这一节把 sed 从"单条替换"升级到"能出结构化改写"：多步一次过、用 `\1 \2` 记住并重排匹配、按"从X到Y"选一段。

---

**1.** In one `sed` pass over `sales.csv`, do **two** edits: drop the header row **and** turn commas into tabs.
*Acceptance:* `sed -e '1d' -e 's/,/\t/g'` → first line `2026-07-01⇥East⇥Widget⇥10⇥2.50` (real tabs).

**2.** Reformat each `passwd.sample` line to `user -> shell` using capture groups.
*Acceptance:* `sed -E 's/^([^:]+):.*:([^:]+)$/\1 -> \2/'` → `root -> /bin/bash`, `daemon -> /usr/sbin/nologin`, …

**3.** Reorder a date `2026-07-01` into `01/07/2026` with three capture groups.
*Acceptance:* `sed -E 's#([0-9]+)-([0-9]+)-([0-9]+)#\3/\2/\1#'` → `01/07/2026`.

**4.** Print the block of `access.log` lines **from** the first `11:00:00` **to** the first `11:03:00` (range between two patterns).
*Acceptance:* `sed -n '/11:00:00/,/11:03:00/p'` → those 4 consecutive lines.

**5.** Transliterate `abc` → `ABC` using `y///` (sed's version of `tr`).
*Acceptance:* `echo "abc" | sed 'y/abc/ABC/'` → `ABC`.

**6.** In `sales.csv`, uppercase `Widget` → `WIDGET` **only on lines containing `East`** (leave other regions' Widget untouched).
*Acceptance:* `sed '/East/ s/Widget/WIDGET/'` → East rows show `WIDGET`; North/West rows still show `Widget`.

**7.** Strip all blank lines from the stream `printf 'a\n\nb\n\n\nc\n'`.
*Acceptance:* `sed '/^$/d'` → `a`, `b`, `c` (no empties).

**8.** Append a new line `INSERTED` **after line 2** of the stream `printf 'a\nb\nc\n'`.
*Acceptance:* `sed '2a INSERTED'` → `a`, `b`, `INSERTED`, `c`.
