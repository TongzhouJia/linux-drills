# KP D5.2 — Mount Network File Systems (NFS)
**6 tasks · RHEL 10 · server1 as client · NFS server = 192.168.55.47**

**1.** Discover which NFS exports are available from `192.168.55.47` and save the list to `/root/exports.txt`.

**2.** Ensure the `nfs-utils` package is installed and any required client services are running.

**3.** Mount the export `/export/dba_files` from `192.168.55.47` at `/mnt/dba_files` (temporary mount).

**4.** Make that NFS mount **persistent** in `/etc/fstab`, using the `_netdev` option.

**5.** Unmount the NFS share, then bring it back with `mount -a`.

**6.** Verify the mount type is `nfs4` and save `findmnt /mnt/dba_files` to `/root/nfs_check.txt`.
