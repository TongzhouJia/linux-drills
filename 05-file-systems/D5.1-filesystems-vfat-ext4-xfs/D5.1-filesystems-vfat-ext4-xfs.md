# KP D5.1 — Create & Mount vfat / ext4 / xfs Filesystems
**6 tasks · RHEL 10 · server2 as root · use spare partitions or LVs**

**1.** Format a partition/LV as **vfat** and mount it at `/mnt/vfat`.

**2.** Format a partition/LV as **ext4** and mount it at `/mnt/ext4`.

**3.** Format a partition/LV as **xfs** and mount it at `/mnt/xfs`.

**4.** Make all three mounts **persistent** in `/etc/fstab` by **UUID**.

**5.** Verify the filesystem type of each mount using `findmnt` and/or `lsblk -f`, and save the result to `/root/fstypes.txt`.

**6.** Unmount `/mnt/vfat`, then confirm all fstab entries still mount cleanly with `mount -a`.
