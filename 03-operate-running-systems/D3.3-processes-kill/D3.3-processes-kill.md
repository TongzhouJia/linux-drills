# KP D3.3 — Identify & Kill Processes
**6 tasks · RHEL 10 · server1 as root**

**1.** Start a background process with `sleep 600 &`, find its PID, then kill it by PID.

**2.** Identify the process currently consuming the **most CPU** and save its command name and PID to `/root/topcpu.txt`.

**3.** Identify the process currently using the **most memory** and save its command name and PID to `/root/topmem.txt`.

**4.** Start three background jobs with `dd if=/dev/zero of=/dev/null &`, then terminate **all of them at once by name** in a single command.

**5.** Find all PIDs of the `sshd` process and save them to `/root/sshd_pids.txt`.

**6.** Start two `sleep 900` processes. Terminate the first **gracefully** (SIGTERM) and the second **forcibly** (SIGKILL). Confirm both are gone.
