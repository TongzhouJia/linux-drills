# KP D7.3 — Restrict Network Access (firewall-cmd / firewalld)
**7 tasks · RHEL 10 · server1 as root**

**1.** Ensure `firewalld` is running and enabled at boot.

**2.** Permanently allow the **http** service in the default zone, then reload.

**3.** Permanently allow **TCP port 8080**, then reload.

**4.** List all rules in the **public** zone and save them to `/root/fw_public.txt`.

**5.** **Remove** the http service permanently and reload.

**6.** Add a **rich rule** that allows **ssh only from `192.168.55.0/24`**.

**7.** Set the **default zone** to a given zone, and assign an interface to a zone.
