# KP D7.1 — Configure IPv4 & IPv6 Addresses (nmcli)
**7 tasks · RHEL 10 · ⭐⭐⭐ per system_info values**

**1.** On **server1**, set a **static IPv4** address `192.168.55.71/24` with gateway `192.168.55.1` and DNS `8.8.8.8` using `nmcli`, and make it persistent.

**2.** On the same connection, set a **static IPv6** address `2002:fe60:def0::55/64`.

**3.** Ensure the connection is set to **autoconnect** at boot.

**4.** Apply the changes (bring the connection down then up) and verify with `ip addr`.

**5.** On **server2**, configure the equivalent: IPv4 `192.168.55.72/24` and IPv6 `2002:fe60:def0::56/64`.

**6.** Add a **secondary** IPv4 address to server1's connection **without removing** the primary one.

**7.** Confirm the configuration persists — save `nmcli connection show <name>` to `/root/nmcli.txt`.
